# フィールドの情報
# フォーマット	：f, [リソース名], [サイズ]

# サイズ		：int型に解釈されるので整数にしてください

# オブジェクトの配置
# フォーマット	：o, [クラス分類], [リソース名], [x座標], [y座標], [z座標], [形状]
# または
# フォーマット	：o, [クラス分類], [リソース名], [x座標], [y座標], [z座標], [形状], [param1]/[param2]...
# クラス分類	：現在 Gimmick, SkinObject, StaticObject, Goalの４つがあります
# リソース名	：上で登録したリソース名を使用します
# 座標			：float型に解釈されるので 1 1.0 1.0f の３つは全て合法です
# 形状			：現在使用していません
# param			：クラスごとに異なります
# Gimmickは animNo / animSpeed / maxFrame を受け取ります
# SkinObjectは animNo / animSpeed / maxFrame を受け取ります
# StaticObjectはパラメータを受け取ります
# Goalはパラメータを受け取りません

o,Goal,goal, 100, -65, 50, null
