■概要
　このデータは、MikuMikuDanceで使用できる「スカイドーム」です。アクセ
サリとして読み込むことで、空や遠景などの背景を作り出せます。
　原理としては単純で、背景となる空などのテクスチャを、球体の内側には
りつけてあるだけです。
　なお、.xファイルですので、対応している他のソフトでも流用できるかも
しれません（その場合は自己責任でお願いします）。

■使用制限
　使用・改変は自由です。特にことわりを入れる必要もありません。使用し
て作成した動画等をネットに投稿される際は、データ置き場のURLか、元動
画のID（ニコニコ動画のID）を表示してくれるとうれしいです。
　データの再配布（改変したものの再配布を含む）も可能ですが、その際に
は、このreadme.txtを添付してください。

■使い方
　アクセサリとして読み込んでください。
　なお、もとの状態でもかなり大きなサイズで作成していますが、マシンスペ
ック等が許す限り大きくする（Siの値を変更すると多きさが変わります）ほう
が、スカイドームの宿命である歪みが出づらくなるでしょう。
　できるかぎりカメラの位置とスカイドームの中心を一致させたほうが、より
自然な見かけになります。

■内容

・黄金色の夕日
　沈みきる前の太陽が金色の輝く瞬間です。
　山あり版と山無し版があります。

・月面
　嘘月面。使いどころ難しすぎかも。

・真っ赤な夕日
　鮮やかな赤に染まった夕日。現実にはちょっとありえない色調だと思います。
おどろおどろしい感じを出すために細い雲を配置してあります。
　山あり版と山無し版があります。

・針の月
　ほそ〜い月です。
　空はうす曇で真っ暗という設定ですんで、ホラー調が似合うかも？
　ちなみに、山を入れても真っ暗で見えないので省略しました。

・昼の月
　よく探すと昼の月があります。
　むしろ、単に雲が多くて靄のかかった空という感じかも。
　山あり版と山無し版があります。

・満月の夜２
　前回の「満月の夜」の、もう少し美しい青を出そうとして失敗しました。単
に月の大人しいバージョンかもしれない。
　山あり版と山無し版があります。

・未知の惑星
　太陽が二つある未知の惑星です。
　どっかで聞いたことあるとかいう話しは抜きで。下が海の理由も追求しない
でください。
